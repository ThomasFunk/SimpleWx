show_dialog()¶

SimpleWx.show_dialog(Name: str) → int | None¶

Shows a predefined generic dialog.

Parameters:

Name (str) – Name of the dialog object. Must be unique.

Returns:

Modal response id, or None in modeless mode.

Return type:

int | None

Examples

response = win.show_dialog(“dialog1”)
