add_tooltip()¶

SimpleWx.add_tooltip(name: str, text: str | None = None) → None¶

Stores and applies tooltip text for a supported widget.

Parameters:

  • name (str) – Widget name key.

  • text (str | None, optional) – Tooltip text to set. If omitted, the stored tooltip is reused.

Examples

win.add_tooltip(“closeButton”, “Closes the application”)
