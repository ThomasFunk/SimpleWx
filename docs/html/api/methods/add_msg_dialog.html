add_msg_dialog()¶

SimpleWx.add_msg_dialog(Name: str, Modal: int = 1, DType: str = 'ok', MType: str = 'info', Icon: str | None = None, RFunc: Callable[[...], Any] | None = None) → None¶

Registers a message-dialog definition for later display.

Parameters:

  • Name (str) – Name of the message-dialog object. Must be unique.

  • Modal (int, optional) – Modal flag (1 modal, 0 modeless).

  • DType (str, optional) – Dialog button type (ok, yesno, okcancel, yesnocancel).

  • MType (str, optional) – Message type (info, warning, error, question).

  • Icon (str | None, optional) – Optional icon path or stock/icon name.

  • RFunc (Callable | None, optional) – Optional response callback (primarily for modeless mode).
