get_treeview()¶

SimpleWx.get_treeview(Name: str, Keyname: str | None = None) → wx.TreeCtrl¶

Returns the underlying wx.TreeCtrl of a TreeView object.

Parameters:

  • Name (str) – Name of the tree-view widget. Must be unique.

  • Keyname (str | None, optional) – Optional compatibility key. Accepted values: list, tree.

Returns:

Native tree-control reference.

Return type:

wx.TreeCtrl
