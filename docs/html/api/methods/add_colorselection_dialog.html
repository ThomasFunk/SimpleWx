add_colorselection_dialog()¶

SimpleWx.add_colorselection_dialog(Name: str, Title: str | None = None, Color: Any | None = None, RFunc: Callable[[...], Any] | None = None) → None¶

Registers a color-selection dialog definition for later display.

Parameters:

  • Name (str) – Name of the color-selection dialog object. Must be unique.

  • Title (str | None, optional) – Dialog title.

  • Color (Any | None, optional) – Initial color as color name, wx.Colour, or [r, g, b] tuple/list.

  • RFunc (Callable | None, optional) – Optional response callback.

Return type:

None.

Examples

win.add_colorselection_dialog(Name=”colorDialog”, Title=”Choose color”)
