simplewx.SimpleWx.get_widget¶

SimpleWx.get_widget(name: str) → wx.Window¶

Retrieves the native wx widget reference for a named entry.

Parameters:

name (str) – Widget name key.

Returns:

Native wx widget object. For DrawingArea, returns the inner drawing panel (not the scroll host container).

Return type:

wx.Window

Examples

button_ref = win.get_widget(“closeButton”) button_ref.Enable(False)
