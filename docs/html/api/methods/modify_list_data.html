modify_list_data()¶

SimpleWx.modify_list_data(Name: str, **commands: Any) → int¶

Modifies list-view row or cell data.

Parameters:

  • Name (str) – Name of the list-view widget. Must be unique.

  • **commands (Any) – Exactly one command mapping, e.g. push=[…], delete=2, set=[row, col, value], clear=None.

Returns:

0 on success.

Return type:

int
